# Weather App

Aplicação web que consome uma API para exibição de dados climáticos em tempo real.

## 🔧 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## ⚙️ Funcionalidades
- Busca de clima por cidade
- Exibição de temperatura, condições climáticas e outros dados
- Consumo de API externa
- Interface responsiva

## 🚀 Acesso ao projeto
👉 Demo: https://clima-app-giliard.netlify.app
👉 Código: https://github.com/giliardolv/clima-app

## 📚 Aprendizados
- Consumo de APIs com fetch
- Manipulação de dados em JavaScript
- Programação assíncrona (async/await)
- Organização de código

## 📌 Status
Projeto funcional, com melhorias planejadas para interface e experiência do usuário.
